class - Bill

attributes -
mealCost (float)
tip (float)
total (float)
is_paid (boolean)
server (string)

methods -
calulate_tip - this method takes a percent of the mealCost and sets it to tip
calulate_total - this method adds up the mealCost and tip to create a total
pay_bill - this method checks if the bill is paid and sets is_paid equal to true or false
pay_server - this method takes the tip and transfers the money to whoever is the server
